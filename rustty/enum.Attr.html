The attributes of a Cell.

Attr enumerates all combinations of attributes a given style may have.

Attr::Default represents no attribute.

Examples

use rustty::Attr;

// Default attribute.
let def = Attr::Default;

// Base attribute.
let base = Attr::Bold;

// Combination.
let comb = Attr::UnderlineReverse;
